## Blog init

https://www.gatsbyjs.com/docs/how-to/previews-deploys-hosting/how-gatsby-works-with-github-pages/

- `npm install -g gatsby-cli`

- `gatsby new "프로젝트명" "테마 경로"`

- `npm i gh-pages`

- `npm run deploy`
